# Rafiq (رفيق) — Premium App Landing Page

A world-class landing page built with React, Tailwind CSS, Framer Motion, and Lenis Smooth Scroll.

## Stack

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion
- React Router DOM
- Lenis Smooth Scroll
- Lucide React / React Icons

## Getting Started

```
npm install
npm run dev
```

## Build

```
npm run build
npm run preview
```
