# About

Rafeeq is a modern Islamic companion designed to deliver a beautiful, distraction-free experience for Muslims around the world.

The application combines elegant UI/UX with reliable Islamic content, accurate calculations, and smooth performance to create a premium daily companion.

---

# Features

- 🕌 Accurate Prayer Times
- 📖 Complete Quran Experience
- 🎧 Quran Audio
- 🤲 Daily Adhkar
- 📿 Duas Collection
- 🧭 Qibla Compass
- 🕌 Ziyarat
- 📚 Sahifa Al-Sajjadiya
- 📖 Mafatih Al-Jinan
- 🌙 Islamic Occasions
- 🎨 Premium UI/UX
- ⚡ Fast Performance
- 🌐 Offline Support
- 🔔 Smart Notifications
